# Decomposer 测试套件

本目录包含针对弹性分解器（ElasticDecomposer）的分级测试套件，按照难度递增分为四个级别：

## 测试级别

### 1. Easy Tests (`easy-tests.js`)
- **目的**: 验证分解器的基础功能
- **测试内容**:
  - 简单任务的解析和分解
  - 结构化任务格式的支持
  - 基础类型标注功能

### 2. Medium Tests (`medium-tests.js`)
- **目的**: 验证分解器的高级功能
- **测试内容**:
  - 多类型任务的语义分组
  - 文件路径冲突的检测与解决
  - 任务间的依赖关系处理

### 3. Hard Tests (`hard-tests.js`)
- **目的**: 验证分解器在复杂场景下的表现
- **测试内容**:
  - 全栈复杂任务处理
  - 模糊/歧义任务处理
  - 大规模重构任务分解

### 4. Integration Tests (`integration-tests.js`)
- **目的**: 验证各功能块的协同工作
- **测试内容**:
  - 不同配置下的表现
  - 插件系统功能
  - 性能基准测试

## 如何运行测试

### 运行所有测试
```bash
node test-runner.js all
```

### 运行特定级别测试
```bash
# 运行简单测试
node test-runner.js easy

# 运行中等难度测试
node test-runner.js medium

# 运行高难度测试
node test-runner.js hard

# 运行集成测试
node test-runner.js integration
```

## 测试验证指标

每组测试都会验证以下指标：

- **功能完整性**: 是否正确生成子任务
- **处理时间**: 各级别任务的处理效率
- **错误处理**: 对异常输入的处理能力
- **冲突解决**: 文件路径冲突的解决效果
- **语义分组**: 相关任务的合理聚合

## 预期输出

测试完成后会输出类似以下的信息：
- 生成的子任务数量
- 处理时间
- 形成的语义组数量
- 冲突解决的数量
- 警告或错误信息

这些测试确保分解器在各种场景下都能可靠地工作。
